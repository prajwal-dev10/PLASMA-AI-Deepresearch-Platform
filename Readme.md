\# 🧠 PRISMA AI – Deep Research



\*\*PRISMA AI\*\* is a multi-agent research assistant that automates the complete research workflow—from planning search strategies to searching the web, generating comprehensive reports, and delivering them via email.



Built using a modular AI agent architecture, each agent is responsible for a specific task, creating an efficient and scalable research pipeline.



\---



\## ✨ Features



\* 🤖 Multi-Agent AI Workflow

\* 🔍 Intelligent Research Planning

\* 🌐 Automated Web Search (DuckDuckGo)

\* 📝 AI-Powered Report Generation

\* 📧 Email Delivery with Resend

\* ⚡ Async Task Execution

\* 🎨 Modern Gradio Interface
