# 🧠 PRISMA AI – Deep Research

PRISMA AI is a **multi-agent AI research assistant** that automates the complete research workflow—from planning search strategies to searching the web, generating detailed reports, and delivering results via email.

 ✨ Features

* Multi-Agent AI Architecture
* Intelligent Research Planning
* Automated Web Search
* AI-Powered Report Generation
* Email Delivery (Resend)
* Modern Gradio Interface
* Asynchronous Processing

 🛠 Tech Stack

* Python
* Google Gemini
* OpenAI Agents SDK
* DuckDuckGo Search
* Gradio
* Resend
* Pydantic
* AsyncIO
